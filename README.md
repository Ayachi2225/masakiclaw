# MasakiClaw
爬取[東雲正树](https://www.zhihu.com/people/Masaki.Ryuu)评论区的所有图片😈

下面是简单介绍，如果想看详细介绍可以看[for chrome](./masakiclaw_chrome/README.md)/[for edge](./masakiclaw_edge/README.md)

## 安装
Chrome和Edge可以直接在插件市场搜索MasakiClaw安装，也可以用[github源码](https://github.com/Ayachi2225/masakiclaw.git)本地构建[在Chrome中安装](./masakiclaw_chrome/README.md#在-chrome-中加载)/[在Edge中安装](./masakiclaw_edge/README.md#在-edge-中加载)，后续计划支持篡改猴脚本
## 模式介绍
主要有三种模式
### 当前页面模式
进入🌲🌲的某个回答/文章/想法，采集评论区

使用方法：进入你想爬取的页面，选择当前页面模式，开始采集即可
### url模式
爬取某个url下的评论区

使用方法：选择url模式，输入你想爬取的url，开始采集即可
### 批量模式
按时间顺序爬取🌲🌲2021-12-30以后的所有回答/文章/想法的评论区的图片。

使用方法：选择批量模式，开始采集即可

**PS:** 我实际上没完整测试过这个模式，所以有bug是正常的（）。

## 各个选项介绍
#### 不限制图片数
  即忽略上面最大图片数的设置，直到爬取完所有图片
#### 允许采集次级评论区图片
  顾名思义即允许采集次级评论区
#### 是否开启查重
开启查重后，会根据哈希值确定相似度，如果重复度超90%则会输出跳过文件保存并输出查重报告
#### 启用ai视觉理解和重命名
支持openai接口的图像识别模型接入，会自动识别图片主要内容重命名文件并生成描述（效果很一般，ai完全get不到梗图的魅力）

为防止api key明文保存在本地，插件使用了ams加密措施，可以使用6位ASCII密码加密，但每回使用该功能要输入密码解密，如果觉得自己的key价值不大或者信任自己的设备可以不设密码。


#### 滚动到底
忽略上面滚动次数限制，一直滚动到底部
