# MasakiClaw

MasakiClaw 是一个 Microsoft Edge MV3 浏览器插件，用于采集知乎评论区图片，并在用户选择的本地文件夹中保存图片和 `images.json` 索引；仅在发现高度相似图片时生成相似图片 Markdown 报告。

插件默认只处理知乎页面，不会把采集结果发送给开发者服务器。AI 视觉理解默认关闭；启用后，图片会发送到用户配置的 OpenAI-compatible 视觉模型 API。

## 在 Edge 中加载

1. 打开 `edge://extensions`。
2. 开启右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择 `masakiclaw_edge` 目录。

## 支持范围

- 当前页模式：仅支持 `https://www.zhihu.com/*` 与 `https://zhuanlan.zhihu.com/*`。
- 指定 URL 模式：仅支持知乎 URL。
- 批量模式：扫描固定 Masaki 用户页中的回答、文章、想法，并进入对应页面采集评论区图片。
- 图片源：知乎图片域名 `zhimg.com`。
- AI 接口：支持 OpenAI-compatible `chat/completions` 视觉模型接口。

## 当前功能

- 当前页模式：采集当前知乎标签页中的评论区图片。
- 指定 URL 模式：打开目标知乎 URL，采集评论区图片，然后关闭临时标签页。
- 批量模式：按日期从近到远扫描 Masaki 的回答、文章、想法，支持停止和图片数量上限。
- 评论区自动识别：自动判断当前页面是否已有内嵌评论区，或是否出现评论弹窗；不再需要手动选择评论区形态。
- 自动滚动：优先定位已加载的评论内容；如果页面没有评论节点，或页面标称评论数大于已渲染评论数，会尝试点击一次“点击查看全部评论/58 条评论”等入口。内嵌评论区扫描会限制在评论容器附近，避免反复滑到页面底部。
- 评论容器滚动：评论窗口存在时优先锁定窗口内部滚动；没有窗口时只扫描页面内嵌评论区，不会在滚动循环中反复点击评论入口。

- 次级评论区：实验选项，默认关闭。勾选后会尝试采集次级回复中的图片。
- 选择文件夹保存：采集完成后打开插件内部写入页，由用户选择本地文件夹；如果目录中已有 `images.json`，则追加新的任务记录。
- 知乎图片处理：优先使用 `data-original`、父级图片链接、`srcset` 大图，并对 `zhimg.com` 常见缩略图 URL 生成原图候选，避免直接下载页面缩略图。
- 表情过滤：会过滤知乎自带 sticker、emoji、reaction、表情、贴纸等资源。
- 知乎状态数据：如果页面的 `#js-initialData` 中已经包含评论实体，插件会优先读取评论对象的 `imageUrl` 和评论内容里的图片。
- JSON 中会保留 `thumbnailUrl` 和 `fallbackUrls`，便于追踪页面缩略图与实际下载候选之间的关系。
- AI 视觉理解：可选填写视觉模型 API Key、Base URL 和模型名；下载页会在保存图片前调用 OpenAI-compatible 接口生成候选文件名和简短描述。API Key 不写入最终 `images.json`。
- 同源 URL 去重：始终开启。规范化原图 URL 相同的图片会在下载前跳过保存，但仍写入 JSON 记录。
- 精细查重：提供一个“是否开启查重”开关，默认关闭。开启后额外使用内容哈希和视觉哈希；内容哈希重复会跳过保存，视觉相似图片仍会保存，并在任务结束后输出简洁重复组总结。
- 最终写入的 `images.json` 只保留任务和图片索引数据；评论定位与滚动诊断只用于开发调试，不写入最终索引。

## AI 与 API Key

- AI 默认关闭。
- 内置常见 AI 域名权限：OpenAI、智谱、SiliconFlow、阿里 DashScope。
- 用户填写其他 HTTPS Base URL 时，Edge 会在首次使用该域名时请求一次运行时权限。
- API Key 默认保存；不勾选明文保存时，必须输入 6 位 ASCII 密码进行本地加密保存。
- 如果不希望保存 API Key，可以取消“保存 API Key”。
- 明文保存必须显式勾选；输入新 API Key 并明文保存时不需要 6 位密码，API Key 会保存到本机 Edge 扩展存储。
- AI 调用失败不会中断采集，图片会按原图名回退保存，并在界面状态和 JSON 中记录失败摘要。

## 隐私说明

- 插件读取用户主动采集的知乎页面评论区内容和图片 URL。
- 采集结果写入用户选择的本地文件夹。
- 插件不会把图片、JSON、相似图片报告或 API Key 发送给开发者服务器。
- 启用 AI 时，图片内容会发送到用户配置的第三方视觉模型 API。
- API Key 仅用于请求用户配置的视觉模型接口，不写入 `images.json`。

## 查重说明

- 规范化原图 URL 相同：始终在下载前判定为确认重复，跳过保存。
- 内容哈希相同：仅在开启查重后计算，下载后判定为确认重复，跳过保存。
- 视觉哈希相似：仅在开启查重后计算，图片仍然保存，并写入 `images.json`；发现高度相似图片时还会生成 Markdown 报告，供用户确认。
- 视觉相似度按 `round((64 - distance) / 64 * 100)` 计算，其中 `distance` 是 64 位视觉哈希的汉明距离。这个百分比只是视觉哈希相似度，不代表 AI 语义置信度。

## 打包 Edge Add-ons 版本

从项目根目录运行：

```bash
./scripts/package-edge.sh
```

脚本会按白名单生成：

```text
dist/masakiclaw_edge_<version>.zip
```

发布包只包含插件运行所需文件，不包含测试 HTML、测试图片或临时文件。
